About the Random Pokemon Generator-Simulator Video Game

This is a full-featured Pokemon battle simulator written in Java. Play with friends on a single computer with splitscreen or duel using online mode.

  • The ultimate Pokemon battle experience!
  • Over 50 Pokemon, with true movesets, abilities, and stats!
  • Colorful and exciting gameplay for beginners or experts!

Pls. No Pokemon Showdown. This does not require a server to play, and can run on any computer!
